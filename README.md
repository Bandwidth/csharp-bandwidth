## Supported Versions
`csharp-bandwidth` should work on all levels of .Net Framework 4.5+. 

| Version | Support Level |
|---------|---------------|
| <4.5 | Unsupported | 
| 4.5 | Supported |
| 4.6 | Supported |
| .Net Core (netstandard1.6)  | Supported |
| PCL | Supported |
| Xamarin (IOS, Android, MonoTouch) | Supported |
